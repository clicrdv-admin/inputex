inputEx - 'inputEg' Example Form Utility

inputEg - normal example rendering
This example a field is rendered normally. There are three visual components
  1. 1. Demo result - either display the rendered Form/Field, or an error message in red
  2. 2. JavaScript - the demo JavaScript that render the demo result
  3. 3. HTML - if the demo support render() and renderDOM(), inputEg will try to render the DOM and print out the result, then clear the demo area and do a real render()
inputEg - error in demo code
This example shows that error message will be displayed in red instead of the rendered content when there is error
